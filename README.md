# 🎂 Whisk & Wish – Gamified Cake Ordering Platform

A gamified cake customization platform where users can design their own cakes by choosing flavors, layers, and themes.

## 🚀 Live Website
🔗 [View the website](https://subaharinig.github.io/wish-and-whisk/)

## 🛠 Tech Stack
- React.js
- Node.js

## ✨ Features
- Cake customization (flavors, layers, themes)
- Gamified experience
- Responsive UI
- Secure order processing
- Real-time order tracking

Initial frontend pushed; backend integration coming soon.
